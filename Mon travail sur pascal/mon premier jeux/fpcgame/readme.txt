Game
====

This is a game, slightly improved from the previous one
to add fullscreen mode and scaling.  Use the following
parameters:

  -fullscreen     fullscreen mode (320x240)
  -double         2x scaling (640x480)
  -triple         3x scaling (960x720)
  -8bit           original 8bit mode

The -8bit parameter is ignored when scaling is used.
